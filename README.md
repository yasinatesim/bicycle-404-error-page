## ⚙️ How To Use

 1. Clone this repository

```bash
git clone https://github.com/yasinatesim/bicycle-404-error-page.git
```

 2. Install the project dependencies
```bash
npm install
```
 3. Develop awesome things
```bash
npm start
```

or

```bash
gulp
```
